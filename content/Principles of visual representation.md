1. Every graphical element (colors, lines, patterns, etc...) should convey `meaning`(information).
	- Should not be used for decoration.
2. Every graphical element `adds weight` to the illustration, use `only` as much weight as you `absolutely need`. 
	- So less [[Cognitive Load Theory|cognitive load]] is needed for the viewer
3. `Whitespace` can be used to `separate` elements or add `emphasis` more `efficiently` than additional weight.
4. [[Every illustration should be intuitive enough to be comprehensible without explanation]].
5. Use `colors` (sparingly) to emphasise/separate specific elements where whitespace isn't enough to accomplish this.
6. `Guide the eye` towards the more `central part(s)`.
