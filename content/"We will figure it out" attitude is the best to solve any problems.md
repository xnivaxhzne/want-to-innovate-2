[[Newer, smaller problems can often appear more daunting initially compared to larger ones already faced]]

- All of us(human beings) are good at something. we can found out a solution for a problem somehow if we work towards it. [[Biologically primary knowledge vs biologically secondary knowledge|Because we are biologically wired for it.]]

- Is there a 100% guarantee for it? 
	- May be not. there could be some % of possibility for slippages.
	- But when we have the **We will figure it out** attitude, we can reduce the stress and the time required to achieve the solution.

- There is a diff between "**I can do anything**" and "**We will figure it out**".
	- In the former, when you get lot of troubles when going for it, you would feel, "ayyoyo, I cant do this". So you might started to feel, if I can't do this, I can't do anything. It is over-positivity which is not good.

- If we have this attitude, we can solve anything easily
